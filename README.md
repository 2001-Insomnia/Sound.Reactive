browser-based, audio-reactive visual instrument built with WebGL2. Microphone input is split into low/mid/high frequency bands and mapped to a grid of tiles, each assigned to one band, producing real-time patterns of light, curves, and lattice interference. 
Mathematical constants (π, φ, e, √2...) drive rotation and pattern frequency, and 12 save slots let you capture and recall favorite settings live.
No audio playback — analysis only, so it's safe to run during a live performance without feedback.
dynamic controls fun to play with. 

The current working version can be run from sisas.neocities.org/audio/Soundreactive
feel free to fork, modify and add whatever to the html. No commercial use allowed.
